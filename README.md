
# 📊 Customer Churn Analysis

## 🔍 Overview
This project focuses on analyzing customer churn behavior in a telecom dataset using Python. The goal is to identify key factors influencing churn and provide actionable business insights.

---

## 📁 Dataset
- Total Records: 7043 customers
- Features include demographics, account details, and service usage
- Target Variable: Churn (Yes/No)

---

## 🧹 Data Cleaning
- Handled missing values in TotalCharges
- Converted data types for analysis
- Transformed categorical variables for better interpretation

---

## 📊 Exploratory Data Analysis (EDA)

### Key Visualizations:
- Countplots for categorical analysis
- Stacked bar charts (percentage-based)
- Churn distribution analysis

---

## 📈 Key Insights:
- 📌 Month-to-month contracts → highest churn (~42%)
- 📌 Electronic check users → highest churn (~45%)
- 📌 Low tenure customers (<1 year) → ~50% churn
- 📌 Senior citizens → higher churn rate

---

## 💡 Business Recommendations:
- Promote long-term contracts
- Improve onboarding experience
- Target high-risk customers early
- Optimize payment methods

---

## 🛠️ Tech Stack:
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 🚀 Conclusion
Customer churn is strongly influenced by tenure, contract type, and payment method. Strategic interventions can significantly improve customer retention.
